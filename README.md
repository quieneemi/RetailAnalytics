# RetailAnalytics
Web app with role-based authorization for crud operations and executing stored functions.

### Run from CLI:
> cd src

> docker compose up

## Libraries & Frameworks
- Frontend:
- - Angular 15 (TypeScript)
- - Angular Materual UI
- Backend:
- - ASP.NET Core
- - Entity Framework Core
- Deployment:
- - Docker
- - Nginx
